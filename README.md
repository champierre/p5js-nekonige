# ネコ逃げゲーム（p5.js版）

マウスから逃げる3匹の猫を操作するゲーム。Scratchの「ネコ逃げ」をp5.jsで再現し、より難しくアレンジしました。

## 機能
- スタートボタンによるゲーム開始
- マウス追跡から逃げる3匹の猫のAI
- 経過時間計測とベストスコア記録
- レスポンシブデザイン
- ゲームオーバー時のリスタート機能

## 遊び方
1. スタートボタンをクリック
2. マウスを動かして3匹の猫に近づける
3. 猫たちが逃げるのを観察
4. 3匹の猫のいずれかに捕まらないように最長時間を目指す
5. ゲームオーバー後はスタートボタンで再開

## 技術要素
- p5.js
- Vanilla JavaScript
- CSS Flexbox
- モダンなUIデザイン
